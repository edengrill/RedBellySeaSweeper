1. Introduction

Welcome to Red Belly Sea Sweeper, a professional audio frequency sweep generator plugin designed for audio testing, room analysis, and creative sound design. This plugin generates precise frequency sweeps with multiple waveform options and sweep modes.

Key Features

  • Frequency sweeps from 20Hz to 20kHz
  • Four waveform types: Sine, Sawtooth, Square, Triangle
  • Three sweep modes: Single, Loop, and Sweep
  • Real-time waveform visualization
  • Keyboard control with X key activation
  • Available in AU and VST3 formats

2. Installation

System Requirements

  • macOS: 10.13 or later
  • Architecture: Intel or Apple Silicon
  • DAW: Any VST3 or AU compatible host

Installation Steps

For AU (Audio Unit):

  1. Copy Red Belly Sea Sweeper.component to:
    ~/Library/Audio/Plug-Ins/Components/
  2. Restart your DAW
  3. The plugin will appear under "Garden of Ed" in your AU plugins list

For VST3:

  1. Copy Red Belly Sea Sweeper.vst3 to:
    ~/Library/Audio/Plug-Ins/VST3/
  2. Restart your DAW
  3. The plugin will appear under "Garden of Ed" in your VST3 plugins list
Tip: If the plugin doesn't appear after installation, try rescanning your plugins in your DAW's preferences.

3. Interface Overview

The Red Belly Sea Sweeper interface consists of:

  • Waveform Display: Real-time visualization of the generated waveform
  • Power Button: ON/OFF switch with red indicator
  • Control Knobs: Three rotary controls for frequency and timing
  • Selector Buttons: Wave type and sweep mode selection
  • Frequency Labels: Display current values below each knob

4. Parameters

Start Frequency

Range: 20Hz - 20kHz
Default: 6.7kHz
Description: Sets the initial frequency of the sweep. In Sweep mode, this is where the sweep begins.

End Frequency

Range: 20Hz - 20kHz
Default: 699Hz
Description: Sets the target frequency of the sweep. In Sweep mode, this is where the sweep ends before reversing.

Sweep Time

Range: 0.1s - 60s
Default: 13.5s
Description: Controls the duration of the sweep from start to end frequency.

5. Basic Operation

Getting Started

  1. Load the plugin on an audio track in your DAW
  2. Set your desired Start and End frequencies
  3. Adjust the Sweep Time
  4. Select a Wave type (Sine is recommended for testing)
  5. Choose a Sweep Mode
  6. Click the Power button or press X to start

Power Control

The plugin can be activated in two ways:

  • Mouse: Click the ON/OFF button
  • Keyboard: Press the X key
Tip: Hover over the power button to see the keyboard shortcut reminder.

6. Keyboard Shortcuts

Key Action
X Start/Stop sweep (toggle power)

7. Sweep Modes

Single Mode

Generates a constant tone at the Start frequency. The End frequency and Sweep Time are ignored in this mode. Useful for testing specific frequencies.

Loop Mode

Continuously sweeps from Start to End frequency, then immediately jumps back to Start and repeats. Creates a sawtooth-like frequency pattern over time.

Sweep Mode (Default)

Sweeps from Start to End frequency, then reverses and sweeps back to Start. Creates a smooth, bidirectional frequency sweep. This is ideal for room testing and speaker analysis.

8. Wave Types

Sine (Default)

Pure tone with no harmonics. Best for accurate frequency response testing and room analysis.

Sawtooth

Rich in harmonics, containing all integer harmonics. Useful for testing harmonic distortion.

Square

Contains only odd harmonics. Good for testing system bandwidth and transient response.

Triangle

Similar to sine but with some odd harmonics. Softer than square wave but richer than sine.

9. Tips & Tricks

Room Analysis: Use Sine wave in Sweep mode with a slow sweep time (20-30 seconds) from 20Hz to 20kHz to identify room resonances.
Speaker Testing: Start with 20Hz to 20kHz sweep to test full-range response, then focus on specific ranges for detailed analysis.
Subwoofer Calibration: Use 20Hz to 120Hz range with a 10-second sweep time to properly calibrate subwoofer crossover and phase.
Creative Uses: Try Square or Sawtooth waves with fast sweep times (0.1-1s) for sci-fi sound effects and transitions.

10. Troubleshooting

Plugin doesn't appear in DAW

  • Verify installation path is correct
  • Rescan plugins in DAW preferences
  • Restart your DAW
  • Check DAW compatibility with AU/VST3

No sound output

  • Check the power button is ON (red)
  • Verify track is not muted
  • Check track routing and output
  • Ensure Start/End frequencies are in audible range

X key doesn't work (VST3)

  • Click anywhere in the plugin window first to give it keyboard focus
  • Some DAWs may intercept the X key - try using the mouse instead

Crackling or distortion

  • Reduce track volume
  • Increase buffer size in DAW settings
  • Close other CPU-intensive applications
